# Proyecto SDAW_3154 - Cristian López Ponce

## Descripción
Este proyecto consiste en una pequeña aplicación web con servidor local utilizando Node.js y su módulo `http`.

## Estructura del proyecto
```
SDAW_3154/
├── index.html
├── script.js
├── server.js
├── package.json
├── .gitignore
└── README.md
```

## Funcionalidad
La página `index.html` incluye un botón que al pulsarlo muestra un mensaje: **"Hola Cristian López Ponce"**.

## Comandos para ejecutar el proyecto
```bash
npm install
npm start
```
Luego, abre tu navegador y visita: [http://localhost:3000](http://localhost:3000)
